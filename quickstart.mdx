## Prerequisites

Before you begin, make sure you have:
- Docker installed on your machine
- Node.js (for AI agent integrations)
- An AI coding assistant (Cursor, Claude, or similar)

## Get started in three steps

### Step 1: Run Insforge locally

```bash
# Clone the repository
git clone https://github.com/insforge/insforge.git
cd insforge

# Set up environment
cp .env.example .env

# Start the backend
docker compose up
```

Your backend is now running:
- **API**: http://localhost:7130
- **Dashboard**: http://localhost:7131

<Note>
  The first run will take a few minutes to download images and initialize the database.
</Note>

### Step 3: Build your first app

Create a new directory for your app (not inside the insforge folder):

```bash
mkdir my-app
cd my-app
```

Now tell your AI assistant to build something:

```
"Build a todo app with user authentication using Insforge as the backend"
```

Your AI will automatically:
- Create database tables
- Set up authentication
- Handle file uploads
- Generate a complete frontend

## Troubleshooting

<AccordionGroup>
  <Accordion icon="circle-exclamation" title="Port already in use">
    If port 7130 or 7131 is already in use, update the ports in your `.env` file:
    ```
    API_PORT=8130
    DASHBOARD_PORT=8131
    ```
  </Accordion>
  
  <Accordion icon="circle-exclamation" title="Docker not starting">
    Make sure Docker Desktop is running and you have sufficient disk space.
  </Accordion>
  
  <Accordion icon="circle-exclamation" title="AI agent not connecting">
    Verify Insforge is running by visiting http://localhost:7130/health
  </Accordion>
</AccordionGroup>
